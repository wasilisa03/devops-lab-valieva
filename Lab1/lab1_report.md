В ходе работы освоила базовые команды Docker для управления контейнерами и образами. Начала с изучения команд docker pull, docker run, docker ps. Поработала с интерактивным контейнером Ubuntu: зашла внутрь, установила пакет curl и проверила его работу. Затем запустила веб-сервер nginx, пробросила порты и проверила доступность в браузере. Научилась смотреть логи, подключаться к работающему контейнеру, останавливать, запускать и удалять контейнеры. Самое важное — разобралась с томами (volumes): создала том, сохранила в нем файл, удалила контейнер, создала новый и убедилась, что файл не пропал.
